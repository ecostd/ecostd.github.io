EcoStd

Standards for Programming Language Tools

What?

We define standards for the tools that make programming tools work in harmony with each other.

Languages

Our initial focus for tool standards is in bringing coherence and interoperation for the tools that drive the C++ programming language.

Getting Involved

We welcome contributions in our efforts for better programming through tools. But until we have a defined contribution process, you can join and inquire in the C++ Ecosystem Evolution forum.
